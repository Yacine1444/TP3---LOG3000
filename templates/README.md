# Module Templates

## Raison du module
Afficher l'interface de l'application web à l'aide des fichiers html. Dans le cas de ce projet, c'est l'interface de la calculatrice.

## Fichiers et leur responsabilités
- **index.html** : Affiche la page centrale de la calculatrice incluant ses boutons, les zones de saisie et le résultat des calculs.

## Dépendances
Les variables et routes établies dans **app.py** et du style de l'interface qui est dans le fichier css **style.css**.
