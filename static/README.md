# Module Static

## Raison du module
Avoir le style visuel et la présentation de l'application à l'aide des fichiers statiques utilisés par la bibliothèque Flask. Dans notre cas, il permet d'avoir l'apparence de la calculatrice web à l'aide du fichier de style.

## Fichiers et leur responsabilités
- **style.css** : Définit le style, les couleurs et la disposition des éléments de l'interface de la calculatrice.

## Dépendances
Aucune dépendance puisque les fichiers css sont appelés depuis le fichier html principal.
