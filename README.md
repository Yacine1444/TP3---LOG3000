# Calculatrice Web – LOG3000  

## Numéro d’équipe  
22  

## Objectif du projet  
L'objectif de ce projet est de créer une application web basique de calculatrice en utilisant la bibliothèque Flask de Python. Cette application offre la possibilité d'exécuter des opérations arithmétiques fondamentales (addition, soustraction, multiplication, division) depuis une interface web.

## Prérequis d’installation
- Un compte GitHub
- Git installé localement 
- Python 
- pip  
 
## Instructions d’installation  
1. **Cloner le dépôt GitHub avec la commande :**  
   ```bash
   git clone https://github.com/<nom-utilisateur>/<nom-du-repo>.git
2. **Se déplacer dans le répertoire du projet :**
   ```bash
   cd <nom-du-repo>
3. **Installer les dépendances avec la commande :**
   ```bash
   pip install -r requirements.txt
4. **Lancer le serveur Flask avec la commande :**
   ```bash
   python app.py
